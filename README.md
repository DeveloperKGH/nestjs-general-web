## Setting the project

```bash
# @nestjs/cli setting
$ npm i -g @nestjs/cli

# Nest 프로젝트 생성
$ nest new project_name

```

## Running the app
```bash
# package.json
$ yarn install

# run
$ nest start

# watch mode
$ nest start --watch
```
